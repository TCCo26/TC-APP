Add a free Finnhub.io key to enable live price refresh on the Portfolio page. News/sports/gaming stay as curated links — those publishers block direct browser API access from static sites.
About the TradingView widgets
The Pulse page uses TradingView's free public embed widgets — they don't support logging into a personal TradingView account; that would require TradingView's paid charting library with server-side auth, which isn't available for regular accounts. The practical equivalent: edit the watchlist symbols directly on the Pulse page (or pull them from your Portfolio holdings with one click) so the widgets track what you actually hold or watch.
